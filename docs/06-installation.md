# Installation

Install Robyroy-V4 manually as a local Codex skill.

```bash
mkdir -p ~/.codex/skills
cp -R robyroy-v4 ~/.codex/skills/robyroy-v4
```

Expected installed structure:

```text
~/.codex/skills/robyroy-v4/
├── SKILL.md
├── contract/
├── scripts/
├── references/
└── evolution/
```

Invoke it explicitly:

```text
$robyroy-v4
```

No global configuration is required by this package.
