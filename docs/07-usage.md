# Usage

## Scoped analysis

```text
$robyroy-v4
Analyze this repository and produce a scoped improvement plan.
```

## Goal Contract before edits

```text
$robyroy-v4
Create a Goal Contract before editing this project.
```

## Controlled documentation update

```text
$robyroy-v4
Run a controlled documentation update with validation and final receipt.
```

## Validation-heavy work

```text
$robyroy-v4
Review this package, run local checks and produce a final receipt with limits.
```

Robyroy-V4 is most useful when the task benefits from scope discipline, local validation and clear reporting.
