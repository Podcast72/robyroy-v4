# Overview

Robyroy-V4 is a governed operator skill for structured local agent work. It helps an agent move from a broad request to a scoped workflow with clear boundaries, local checks and a final receipt.

The problem is simple: agent work often starts with a natural-language request, but complex tasks need more than improvisation. A prompt can describe the desired behavior, yet the behavior can drift as the task grows.

Robyroy-V4 packages the workflow as a skill. It gives the agent a repeatable operating model built around modes, a Goal Contract, scope checks, contract files, event logs, controlled mini-prompts and validation scripts.

The result is not a global security system. It is an experimental local operator pattern that makes agent work easier to review, continue and improve.
