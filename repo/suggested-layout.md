# Suggested Layout

The public repository should keep the installable skill files at the root and place explanatory material in `docs/` and `examples/`.

```text
robyroy-v4/
├── SKILL.md
├── contract/
├── scripts/
├── references/
├── docs/
├── examples/
├── assets/
└── repo/
```

Runtime logs should remain local and should not be committed.
